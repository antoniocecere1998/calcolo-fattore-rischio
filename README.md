# Calcolo del fattore di rischio – Project Work

Questa repository contiene il materiale sviluppato per il project work relativo al calcolo del fattore di rischio in ambito di sicurezza sul lavoro.

## Contenuto della repository

La repository include i seguenti file:

- `index.html`  
  Pagina web principale per il download e la consultazione del documento PDF.

- `styles.css`  
  Foglio di stile CSS utilizzato per la presentazione grafica della pagina, separato dalla struttura HTML.

- `Documento_calcolo_fattore_rischio.pdf`  
  Documento esplicativo sul calcolo del fattore di rischio, comprensivo di inquadramento normativo, metodologico ed esempi applicativi.

- `Rapporto_sviluppo_e_test.pdf`
  Documento di supporto che descrive le fasi di sviluppo dei codici HTML/CSS,strumenti/librerie usate,e una sintesi dei test di controllo di accessibilità di base e prova su dispositivi mobili.
